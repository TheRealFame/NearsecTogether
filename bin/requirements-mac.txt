# NearsecTogether macOS Requirements
#
# MANDATORY:
#   - Node.js (v18+): https://nodejs.org/
#   - Python 3.8+: https://www.python.org/
#   - npm: comes with Node.js
#
# FOR KBM SUPPORT (Keyboard/Mouse only):
#   - pyautogui Python package: installed by pip (see below)
#
# FOR GAMEPAD SUPPORT:
#   - NOT SUPPORTED on macOS via NearsecTogether
#   - macOS has no equivalent to Linux uinput or Windows ViGEmBus
#   - Workaround: Use native macOS gamepad support in games directly
#   - Alternative: Install third-party tools like Karabiner or ControllerMate
#
# SETUP INSTRUCTIONS:
#   1. Install Node.js and Python 3 (e.g., via Homebrew or direct download)
#      brew install node python@3.11
#   2. Extract NearsecTogether to a folder
#   3. Open Terminal in the NearsecTogether folder
#   4. Run: pip3 install -r bin/requirements-mac.txt
#   5. Run: npm install
#   6. Run: ./bin/start.cmd  OR  npm run electron
#
# KNOWN LIMITATIONS:
#   - Gamepad injection is NOT supported on macOS (no system API)
#   - Only keyboard/mouse passthrough available (via pyautogui)
#   - Motion controls not supported
#   - Audio playback uses native afplay command
#
